## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

## Features
- Authentication system
- Dashboard with tile management
- Category management
- Color management
- Submission tracking

## Tech Stack
- React
- React Router
- Tailwind CSS
- Context API for state management
